# Analisador de Embarques .txt (Datasul)

Ferramenta profissional para extração, análise e consolidação de relatórios de embarque do ERP Datasul em formato texto.

## Funcionalidades

- **Processamento Local:** Leitura rápida diretamente no navegador, sem tráfego de dados para servidores externos.
- **Resumo Executivo:** Visualização instantânea de Totais de Embalagens, Peso (kg) e Cubagem (m³).
- **Filtros Inteligentes:** Navegação facilitada por número de embarque.
- **Relatório Consolidado:** Emissão de documento simplificado por pedido, otimizado para conferência e impressão.
- **Exportação Multiformato:** Suporte para planilhas XLSX (Excel) e arquivos CSV.
- **Rastreamento de Inconsistências:** Sistema de log para identificação de linhas com falha na leitura.

## Como Utilizar

1. Abra o `index.html` em qualquer navegador moderno.
2. Arraste e solte o arquivo `.txt` do relatório na área de upload.
3. Utilize o painel de filtros para navegar pelos embarques.
4. Gere relatórios de impressão ou exporte para planilhas conforme a necessidade.

## Especificações Técnicas

- Desenvolvido em Vanilla JavaScript, HTML5 e CSS3 moderno.
- Dependência externa: SheetJS (carregada via CDN) para gestão de planilhas.
- Design responsivo com tipografia Inter.
